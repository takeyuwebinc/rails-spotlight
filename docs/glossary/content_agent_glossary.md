# 掲載内容管理支援エージェント 用語集

対象範囲: 掲載内容管理支援エージェント（`ContentAgent` 名前空間）のドメイン用語。
種: 同機能の要件定義書「6. 用語定義」セクション（実装完了に伴い削除済み。git 履歴で参照可能）。

## 凡例

| 項目 | 説明 |
|------|------|
| 用語 | 正式な用語名（プロジェクト内で統一して使用する名称） |
| 定義 | 用語の意味を簡潔かつ正確に記述 |
| 別名 | 許容される同義語（正式用語の使用を推奨） |
| 禁止表現 | 混乱を招くため使用を避ける表現（括弧内は理由） |
| 英語名 | 対応する英語表記（モデル名・カラム名・enum 値等、コード上の命名の基礎） |
| 関連用語 | 併せて参照すべき用語 |

## ドメイン

| 用語 | 定義 | 別名 | 禁止表現 | 英語名 | 関連用語 |
|------|------|------|----------|--------|----------|
| 掲載内容 | サイトに公開する 4 種のコンテンツ（Project / SpeakingEngagement / UsesItem / Slide）の総称。エージェントが操作できる対象はこの 4 種のみ | — | コンテンツ（範囲が曖昧） | `PendingChange::TARGET_TYPES` | 保留変更 |
| エージェント | アプリに組み込まれた、ツール実行能力を持つ LLM ベースの対話プログラム。会話進行エージェントが本体で、さくらのAI Engine のモデルで動く | 会話進行エージェント | AI（範囲が曖昧）、bot | `ContentAgent::ConversationAgent` | ツール、下位タスク |
| 会話 | 管理者とエージェントの一連のやり取りの単位。DB に無期限保存され、一覧から再開できる。会話の削除機能は提供していない（スコープ外） | チャット | セッション（HTTP セッションと混同） | `Chat` | メッセージ、コスト概算 |
| 保留変更 | エージェントが提案した掲載内容への変更。承認されるまで掲載内容と隔離して保持される。承認ゲートの中核（ADR: SPOTLIGHT-RAILS-31） | 提案 | 下書き（掲載内容側の非公開状態と混同） | `ContentAgent::PendingChange` | 承認、適用処理、プレビュー |
| プレビュー | 保留変更の内容を画面表示したもの。更新・公開切替では対象レコードの現在値を表示時に取得して新旧比較する（保留変更に旧値は保存しない） | プレビューカード、登録案 | — | （View パーシャル `_pending_change`） | 保留変更 |
| 承認 | プレビューを確認した管理者が適用を許可する明示的な画面操作。会話上の同意表明は承認ではない | — | 了承・合意（会話上の表明と混同） | `PendingChangeApprovalsController` | 適用処理、否認 |
| 適用処理 | 承認された保留変更を掲載内容へ反映する唯一の入口。保留変更の payload をそのまま反映し、プレビューとの同一性を保証する | — | 実行（ツール実行と混同） | `ContentAgent::ApplyPendingChange` | 保留変更、承認 |
| 読み取り系ツール | 掲載内容を検索・一覧・参照するツール。DB を変更しないため承認なしで自動実行される。「検索」は一覧ツールのキーワード絞り込みで実現する | — | — | `ListContentsTool` / `GetContentTool` | 外部取得系ツール |
| 外部取得系ツール | URL 取得・Web 検索によりアプリ外部から情報を取得するツール。DB を変更しないため承認なしで自動実行される。取得内容は属性抽出の素材としてのみ扱う（プロンプトインジェクション対策は承認ゲートが担う） | — | 読み取り系（DB 参照と外部アクセスを区別するため） | `FetchUrlTool` / `WebSearchTool` | 承認 |
| 書き込み系ツール | 掲載内容への変更を提案するツール。掲載内容の DB は変更せず保留変更を作成するのみ | 提案ツール | — | `ProposeChangeTool` | 保留変更 |
| 下位タスク | 会話進行の外で別モデルにより実行される LLM 処理（属性抽出・検索結果要約）。利用トークンは下位タスク利用量として記録し会話コストに含める | サブタスク | — | `ContentAgent::TaskUsage`（記録）/ `ExtractAttributesTool` | コスト概算 |

## 運用上の注意

- 掲載内容の公開・非公開は、Project / Slide では `published_at`（未来日時=非公開）、SpeakingEngagement / UsesItem では `published` フラグで表現する。エージェントの新規作成提案には公開状態が必須（`PendingChange` のバリデーション）
- タスク別モデル割当と単価表は静的設定（`ContentAgent::MODEL_ASSIGNMENTS` / `ChatCost::PRICING_PER_10K_TOKENS`）。さくらのAI Engine の単価改定時は単価表のみ更新する
- llm-jp-3.1-8x13b-instruct4 は tool calling 非対応（基盤側制限）のため会話進行に割り当てない（ADR: SPOTLIGHT-RAILS-29）
